a와 b 출력하기

[문제설명]
정수 a와 b가 주어집닌다. 각 수를 입력받아 입출력 예와 같은 형식으로 출력하는 코드를 작성해보세요.

[제한사항]
-100,000 <= a,b <= 100,000

[결과값]
a = 4
b = 5



import java.util.Scanner;

public class Solution{
	public static void main(String[] args)
}

		Scanner sc = new Scanner(System.in);

	int a = sc.nextInt();
	int b = sc.nextInt();
 
	System.out.println("a = " + a + "\n" + "b = " + b);
	
	}
}
